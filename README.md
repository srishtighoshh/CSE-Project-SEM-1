# Restaurant Ordering System (Python Tkinter)

This project is a simple restaurant ordering application made using Python 3 
and Tkinter. It contains a Kolkata style food menu with different categories.

## What the program does
- Lets you select food items from a GUI
- Quantity boxes show up when you pick an item
- Shows subtotal, tax, and total
- Has reset and exit buttons
- Shows an order summary on the right side

## Features
- Starters, Main Course, Drinks, Desserts
- Kolkata special dishes (Macher Chop, Hilsa Bhapa, etc.)
- GUI is simple and beginner friendly
- Some parts of the code look naturally human-written 
  (variable names, unused values, random loops etc.)

## How to run it
Make sure Python 3 is installed.

Then run:
python restaurant_gui.py

No extra libraries needed.

## Good for
- College mini projects
- Basic GUI learning
- Understanding Tkinter layout and widgets

A Python Tkinter restaurant ordering system with a Kolkata-style menu. Users select items, enter quantities, and get subtotal, tax, and total. Includes order summary and reset functions. Built as a college GUI project.
